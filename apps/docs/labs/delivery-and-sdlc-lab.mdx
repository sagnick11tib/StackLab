# Delivery and SDLC Lab

This lab covers how software moves from local work to stable production systems.

## Coverage Areas

- local development workflows
- CI pipelines
- artifact build flow
- testing strategy
- deployment progression
- rollback strategy
- config management
- feature-flag rollout

## Desired Outcome

Users should understand how engineering process choices affect:

- delivery speed
- release safety
- production recovery
- developer experience
