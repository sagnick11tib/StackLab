# FAQ

<AccordionGroup>
  <Accordion title="Is StackLab only a docs site?" defaultOpen>
    No. The docs site is the canonical public surface today, but the broader product includes interactive labs, comparisons, applied scenarios, and production templates.
  </Accordion>
  <Accordion title="Is StackLab only for system design interviews?">
    No. Interview preparation is one audience, but the broader goal is practical engineering understanding across frontend, backend, data, reliability, and delivery.
  </Accordion>
  <Accordion title="Why use Mintlify first?">
    Mintlify gives StackLab a structured, searchable, AI-friendly public source of truth before the interactive product surface exists.
  </Accordion>
  <Accordion title="Are contributor and AI docs private?">
    No. They are public but hidden from the main navigation so they remain accessible without dominating the public docs experience.
  </Accordion>
</AccordionGroup>
