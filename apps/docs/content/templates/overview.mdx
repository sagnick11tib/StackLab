# Templates Overview

Templates are one of StackLab's practical outputs. They should help users move from learning to building without relying on weak defaults.

## Template Principles

- production-oriented defaults
- clear intended use case
- architecture rationale
- explicit tradeoffs
- safe extension guidance
- documentation that explains what is intentionally omitted
