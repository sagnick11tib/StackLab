# Applied Architecture Lab

Applied labs connect multiple concepts into product-shaped systems.

## First Scenario Families

- booking systems
- chat systems
- payment flows
- notifications pipelines
- file upload and processing
- e-commerce paths

## What Makes These Labs Valuable

They expose the difference between:

- a concept in isolation
- and a concept inside a real product with constraints, concurrency, failure, and growth

## Standard Questions

- what constraints matter here
- what is correctness-sensitive
- what is eventually consistent
- what is the user-facing failure mode
- how would the architecture evolve at higher scale
