# AI Agent Guide

This page mirrors the public-safe parts of `AGENTS.md`.

## Mission

StackLab is an open-source software engineering platform focused on production concepts, visual simulations, pattern comparison, and production-grade templates.

## Working Rules

- treat `apps/docs` as the canonical docs content source
- treat the Nextra docs frontend as the primary public docs destination
- keep repo-local operational guidance in `AGENTS.md`
- do not rewrite the product into a different category
- do not introduce generic template aesthetics that conflict with the StackLab design direction
- do not add features that do not fit the product pillars

## Product Pillars

- Learn
- Simulate
- Compare
- Build
- Contribute

## Documentation Rule

If a change affects product direction, canonical docs content should be updated in `apps/docs` so both public frontends stay aligned.
