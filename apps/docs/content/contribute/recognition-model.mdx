# Recognition Model

Contributor recognition matters because StackLab wants people to help build the platform with care.

## Planned Recognition Mechanisms

- contributor wall
- module-level credits
- template credits
- contributor spotlight features
- recognition for major documentation and architecture improvements

## Recognition Rule

Recognition should reward:

- quality
- clarity
- product fit
- maintainability

not just volume of merged changes.
